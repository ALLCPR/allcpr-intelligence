      About & data tiers

      How to use it
      • Historical ALLCPR = real Enrollware/class history. Course buttons show AHA BLS / ARC BLS / ARC CPR student counts.
      • Modeled baseline = public-data estimate from Census ACS + Gazetteer, for every US ZIP. Course buttons show demand tilts (AHA BLS & ARC BLS = healthcare-workforce BLS demand; ARC CPR = community/layperson CPR demand) — the modeled layer estimates demand type, not AHA-vs-ARC brand preference.
      • Enriched modeled = modeled baseline + Places / Routes / competitor / commercial validation. Enriched validation available for priority ZIPs.
      • Modeled score = public-data estimate. Historical score = real ALLCPR evidence.
      • Smooth heat = regional intensity, not exact ZIP boundaries. The ZIP detail panel is exact ZIP-level evidence.
      • Google Places is context enrichment only and is not called on page load.
      • Search a ZIP or click the map to inspect it; add ZIPs to Compare. Treat recommendations as test/validation guidance, not a lease guarantee.
      Data tiers & quality
      • Real history measured Enrollware outcomes.
      • Modeled public Census estimate (baseline).
      • Enriched priority ZIPs with Places/Routes/competitor/commercial validation.
      • Commercial validated a manually-checked, available space exists.
